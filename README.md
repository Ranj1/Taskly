# üìù Taskly - Flutter ToDo App

**Taskly** is a beautiful and easy-to-use to-do list application built with Flutter.  
This app helps you add, manage, and filter your daily tasks with **BLoC state management** and **SQLite** local database.

---

## ‚ú® Features

‚úÖ Create, edit, and delete tasks  
‚úÖ Prioritize tasks as **Low**, **Medium**, or **High**  
‚úÖ Filter tasks by **All**, **Completed**, or **Pending**  
‚úÖ Swipe to delete tasks  
‚úÖ Light & Dark theme support with a toggle  
‚úÖ Store all tasks in a local **SQLite** database  
‚úÖ Smooth animations using **Lottie**
